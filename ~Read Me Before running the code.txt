1. Open Command Prompt (CMD).
2. Type the following commands:
	pip install flask
	pip install flask-login
	pip install flask-sqlalchemy
	pip install Flask_mail
	pip install Requests
3. Go to the following link to download python https://www.python.org/downloads/
4. open this folder in Visual Studio code (VS Code)
5. Run 'main.py' file
6. click on the local address for the website shown in the terminal after running 'main.py'
	The local address could somewhat look like 'http://127.0.0.1:500'
7. OR simply you can have a look at the website by visiting 'bookmyspot.onrender.com'
	after typing the above URL, wait 50-60 seconds for the backend onrender web server to start.
